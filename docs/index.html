📋 Shared Clipboard

Aynı Wi-Fi'daki cihazlarda ortak pano. Birinde kopyala, hepsinde yapıştır.

Sürüm yükleniyor…
    Mac sürümü yalnızca Apple Silicon (M1/M2/M3/M4) için derleniyor. Intel Mac şu an desteklenmiyor.

    Nasıl çalışır?

    Menü çubuğunda / sistem tepsinde çalışır. Aynı Wi-Fi'daki her cihaz kendini mDNS (Bonjour) ile yayınlar; cihazlar birbirine doğrudan WebSocket'le bağlanır. Ortada sunucu yok, hesap yok, veri yerel ağdan çıkmıyor.

    İlk açılışta yapılacaklar

    macOS — "Açılamıyor" / "damaged" uyarısı
    1. ZIP'i aç → SharedClipboard.app'ı /Applications klasörüne taşı.
    2. Sağ tık → Aç → açılan dialog'da tekrar Aç.
      (Çift tıklama ilk seferde çalışmaz; bu yapı imzalı değil.)
    3. "SharedClipboard is damaged and can't be opened" diyorsa Terminal'i aç ve şu komutu yapıştır: 
      xattr -dr com.apple.quarantine /Applications/SharedClipboard.app
      Sonra normal çift tıkla aç.
    4. macOS 14+ ise "…yerel ağındaki cihazları bulmak istiyor" izni çıkacak → İzin Ver.
      Kaçırırsan: Sistem Ayarları → Gizlilik & Güvenlik → Yerel Ağ → SharedClipboard'u aç.
    Windows — SmartScreen ve Firewall
    1. ZIP'i aç. SharedClipboard.exe'yi çalıştır.
    2. Mavi SmartScreen ekranında Daha fazla bilgi → Yine de çalıştır.
      (İmzalı değil; bu nedenle uyarı çıkıyor.)
    3. Windows Defender Firewall iki kez izin soracak (mDNS keşfi ve WebSocket portu için) → Özel ağlar'ı işaretle ve İzin Ver.
    Linux — clipboard araçları
    1. X11: sudo apt install xclip python3-tk libappindicator3-1
    2. Wayland: sudo apt install wl-clipboard python3-tk
    3. İndirilen dosyayı çalıştırılabilir yap: chmod +x SharedClipboard

    Güvenlik notu

    Trafik yerel ağda şifresiz akıyor (TLS yok). Bu uygulamayı yalnızca güvendiğin ev/ofis Wi-Fi'larında kullan. Açık veya halka açık ağlarda çalıştırma.
